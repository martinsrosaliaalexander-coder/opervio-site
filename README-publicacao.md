# Opervio Solutions — Site institucional

Arquivos prontos para publicação no GitHub Pages.

## Conteúdo
- `index.html`
- `assets/logo-opervio.png`
- `assets/icone-opervio.png`
- `CNAME`
- `.nojekyll`

## Publicação
1. Suba os arquivos para um repositório público no GitHub.
2. Vá em Settings > Pages.
3. Escolha Deploy from a branch.
4. Selecione a branch `main` e a pasta `/root`.
5. Confirme o domínio personalizado `opervio.com.br`.
